# camelcase-keys

[![npm version](https://img.shields.io/npm/v/camelcase-keys.svg)](https://www.npmjs.com/package/camelcase-keys)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Convert object keys between camelCase, snake_case, and kebab-case

## Installation

```bash
npm install camelcase-keys
```

## Functions

- `toCamelCase` — Convert string to camelCase
- `toSnakeCase` — Convert string to snake_case
- `toKebabCase` — Convert string to kebab-case
- `camelizeKeys` — Convert all object keys to camelCase
